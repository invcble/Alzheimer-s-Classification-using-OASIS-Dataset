The link for original dataset is https://www.kaggle.com/datasets/ninadaithal/imagesoasis

This project was done is Google Colab, and data being so huge, in our code it is being uploaded through GDrive. Here are the steps to run our code.

1. Download the dataset, upload to GDrive.

2. Run the first cell of the Python code, mount GDrive as local drive.

3. Copy the path of Dataset.zip from left side file section and paste it into unzip code.

4. Run the code.
